### 개발 인원
1명

### 프로젝트의 목적

뉴스에서 소방복을 일반세탁기로 돌리는 것을 봤습니다.
근데 일반 세탁기로 돌리면 방화복의 원래 기능을 잃을 수 있다고 합니다.
그런 행동이 반복되면 소방관들이 위험해 질수 있으니 방화복 관리 시스템을 계획하게 되었습니다.

소방관들의 방화복의 상태를 한눈에 볼 수 있는 서비스
